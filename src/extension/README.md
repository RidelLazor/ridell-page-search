# RidelL Chrome Extension

This folder contains a Chrome extension that replaces your new tab page with RidelL.

## Features

- **Search**: Quick access to web search directly from the new tab
- **Customizable Shortcuts**: Add, edit, and remove shortcuts to your favorite sites (up to 10)
- **Google Apps Grid**: Quick access to popular Google services
- **Theme Options**: Light, Dark, and System themes
- **Accent Colors**: Choose from 6 accent colors (Blue, Rose, Orange, Green, Purple, Cyan)
- **Custom Fonts**: 6 font options (Inter, Space Grotesk, Poppins, JetBrains Mono, Outfit, Sora)
- **Background Options**: Default, Gradient, Mesh, Dots, Aurora (animated), Sunset (animated), Ocean (animated), or custom image URL
- **Persistent Settings**: All customizations are saved to local storage

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" in the top right
3. Click "Load unpacked"
4. Select the `src/extension` folder from this project
5. Open a new tab to see RidelL!

## Customization

Click the "Customize RidelL" button in the bottom-right corner to access:

- **Theme**: Switch between Light, Dark, or System (auto)
- **Accent Color**: Choose your preferred accent color
- **Font**: Select from 6 different font families
- **Background**: Choose from 8 background options including animated gradients

## Shortcuts

- Click on any shortcut to navigate to that site
- Hover over a shortcut to see the edit button
- Click the edit button to modify or remove a shortcut
- Click "Add shortcut" to add a new site (up to 10)
